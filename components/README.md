# components/

业务组件，按业务域拆分。

## 目录

| 目录 | 业务域 |
|---|---|
| `agent/` | Agent 工作流展示、状态、动画 |
| `payment/` | Payment Plan、付款列表、金额展示 |
| `risk/` | Risk Check、预算检查、白名单检查、限额检查 |
| `caw/` | Cobo Agentic Wallet Execution、交易状态、tx hash |
| `audit/` | Audit Report、可审计记录展示 |
| `landing/` | 首页 Landing、Hero、Feature 展示 |
| `shared/` | 跨业务域共享组件 |
| `ui/` | 纯 UI 组件（Button 变体、Card 变体等） |
