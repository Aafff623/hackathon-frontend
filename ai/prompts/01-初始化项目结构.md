# Prompt: 初始化项目结构

## 目标

初始化 AgentCFO 前端项目的工程骨架、维护文档、Mock 数据、Handoff 文档和工具链状态文档。

## 输入

- 项目背景（见 ai/memory/01-项目上下文.md）
- 目录结构规范（见 CLAUDE.md）

## 执行步骤

1. 读取当前目录状态
2. 检查工具链状态
3. 创建目录结构
4. 创建根规范文件（CLAUDE.md, AGENTS.md, README.md, .env.example）
5. 创建 app 占位文件
6. 创建 components 结构
7. 创建 lib（api / mock / workflow / constants）
8. 创建 docs
9. 创建 handoff
10. 创建 ai/memory / prompts / checklists
11. 创建 assets / public
12. 输出初始化报告

## 约束

- 不安装未知依赖
- 不写入任何密钥
- 不覆盖已有文件
- 如果不是 Next.js 项目，只创建占位结构
