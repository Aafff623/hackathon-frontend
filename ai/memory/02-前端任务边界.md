## Mock-first 策略

- `NEXT_PUBLIC_DEMO_MODE=mock` 是默认演示模式
- 真实 API 尚未接入
- `lib/api/` 当前应视为 API adapter 层，不允许组件直接 `fetch`
- 组件必须通过 adapter 获取数据（当前走 `lib/mock/`）
- 后续联调时，在 `lib/api/payment.ts`、`lib/api/caw.ts` 中接入真实 `request()`
