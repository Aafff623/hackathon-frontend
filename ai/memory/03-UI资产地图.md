# UI 资产地图

## HeroUI Pro

- 状态：Skill 已安装，MCP 已连接
- 用途：Dashboard / Form / Card / Modal / Table / Badge / Status / KPI / DataGrid / Sheet / Command / Audit Report
- 规则：使用 compound API，关注 aria-label / textValue / getRowId，使用 design tokens

### 复杂组件 checklist

#### DataGrid
- 必须有 `aria-label`
- columns 每列必须有 `id`
- 使用 `accessorKey` 映射字段
- 必须有 `getRowId`
- loading / empty / error 状态必须设计

#### Kanban
- `Kanban.Card` 必须通过 `Kanban.CardList` render prop 渲染
- Card 必须有 `id` 和 `textValue`
- CardList 必须有 `aria-label`
- 不要直接 `Array.map` 渲染 Card

#### Sheet
- 用于 approval / payment detail / audit detail 等确认型交互
- 必须有明确关闭路径
- 不能隐藏关键风险信息

#### Command
- 用于后续快捷操作，不作为 MVP 必需
- `Command.Item` 需要 `textValue`

#### 通用状态
所有复杂展示组件必须设计：loading / empty / error / success / blocked / pending approval
