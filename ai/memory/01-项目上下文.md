# 项目上下文

## 项目

AgentCFO｜DAO AI 财务官

## 定位

面向 Web3 小团队 / DAO 的 AI 财务官。

## 核心 Demo 流程

```txt
贡献记录 / 订阅账单输入
→ AgentCFO 解析任务
→ 生成 Payment Plan
→ Risk Check：预算 / 白名单 / 单笔限额 / 重复付款
→ Human Approval
→ Cobo Agentic Wallet 执行付款
→ 展示 tx hash / Agent Wallet / Payment Status
→ 生成 Audit Report
```

## 技术栈

Next.js + React + TypeScript + Tailwind CSS + HeroUI v3/Pro + Aceternity

## 前端范围

- 项目首页（Landing）
- Demo Console
- 贡献记录输入
- Agent 工作流展示
- Payment Plan 展示
- Risk Check 展示
- Human Approval
- CAW Execution 展示
- tx hash / Agent Wallet 展示
- Audit Report
- Mock fallback
