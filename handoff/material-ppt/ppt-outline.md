# PPT Outline

## 1. 封面

- 项目名称：AgentCFO｜DAO AI 财务官
- 赛道：Cobo Agentic Wallet / Agentic Commerce
- 一句话定位：面向 Web3 小团队 / DAO 的 AI 财务官

## 2. 问题

- Web3 小团队 / DAO 的财务流程碎片化
- 贡献记录、预算管理、付款执行缺乏统一工具
- 人工审批效率低，容易出错

## 3. 解决方案

- AgentCFO：AI 驱动的财务官
- 自动解析贡献记录 → 生成 Payment Plan → Risk Check → 人工确认 → CAW 执行 → Audit Report

## 4. 核心流程

```txt
贡献记录输入
→ Agent 解析
→ Payment Plan
→ Risk Check（预算 / 白名单 / 限额 / 重复）
→ Human Approval
→ CAW Execution
→ Audit Report
```

## 5. 技术亮点

- Cobo Agentic Wallet 集成
- 权限边界内自动执行
- 完整可审计链路
